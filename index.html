How to Create GITHUB-PAGES

With in 5 Steps one can create a simple/Basic Github page
Stetp 1
First Create a new repository (Ignore this if you want to create github-page for existing project)
Step 2
Github Pages will look for either Index.html or README.md(if not Index.html) file for page content from a mapped branch and one can use only master branch or gh-pages(a special branch) are allowed to select. In this demo I am going to create gh-pages branch (you can use master branch also)
Step 3
We almost configured all the things need for github pages. its time to configure github page. Open setting of your repository and scroll down to the section Github pages.Click on the option button and Choose the branch. Bingo your github page is ready for the project
Step 4
Currently github page is accessing content from README.md, In order to change that we are going to create Index file in the project main directory and added the html content need for your porject. Now go and refresh the github page to see the changes (sometimes it will take time to refreh)
Step 5
Mentioned the Gihubpage rule of a project in its README.md to file to get easy access to the Page. Thank you
